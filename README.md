# psf
package static file single go file easy to use

# Requirement
go 1.8

# Build
```
go build psf.go
```

# Usage
```
Usage of psf:
--path                  static file path, more than one , separated.
--name                  save go file name, suffix .go is not required, default is psf_resource
```
